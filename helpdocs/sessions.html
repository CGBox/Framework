Sessions

Sessions are easier now with our new session handling method.
In order to set sessions, we must do this:
$this->session->set_userdata();
Now you can put an array though there, or a normal value. We will sample both.
// Array
$this->session->set_userdata(array('session_name' => 'session_value'));
// Normal
$this->session->set_userdata('session_value', 'session_name');

How to check if session is active

if($this->session->userdata('session_name')) { echo 'session active'; } else { echo 'no active session'; }
How to destroy all sessions
$this->session->dc();
How to destory one session
$this->session->dco('session_name');
