Library

We use a library system, meaning anyone can import / create librarys and share/use them. In this example we create a simple library.
We start out by creating a file in application/librarys/ and I name it personally: example.php(you can name it anything you want).
Now, we have to create a class for the library, the library class will extend the Library itself.
<?php
class Example extends Library {
function __construct() {
parent::__construct(); // Parent the constructor
}
public function run() {
// To be called when run.
echo 'We made it!'; // Display we made it message.
}
}
?>

Now to execute this code, we open our MODEL (where we will be calling it) and run:
$this->library->load('example');
// Then we can do:
$this->library->example->run();

And that is how we created our very first Library.

This framework is opensource and free.
